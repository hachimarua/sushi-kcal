# Vol.0.3 iPhone 実機確認チェックリスト

## ホーム画面追加

- iPhone Safari でアプリURLを開く
- 共有メニューから「ホーム画面に追加」を選ぶ
- ホーム画面アイコンが、まぐろ・たまご・ガリのアイコンになっている
- 表示名が「すしkcal」になっている
- ホーム画面から起動したとき、Safariのアドレスバーなしで開く

## 起動直後

- 「いらっしゃいませ!」画面が表示される
- スシロー / くら寿司の選択が片手で押せる
- 文字が詰まらず、のれん背景が邪魔にならない
- 背景色が MealTracker と被りすぎない

## 商品検索

- 検索欄をタップしても iOS の自動ズームが起きない
- `まぐろ`、`いか`、`茶碗蒸し` で候補が出る
- 候補1件の高さが十分あり、誤タップしにくい
- 候補をタップすると「今日の皿」に追加される

## 今日の皿

- 皿レーンが皿として認識できる
- 1皿、3皿、6皿程度で表示が破綻しない
- `+` / `-` が片手で押せる
- リセットで確認ダイアログが出る

## MealTracker転記

- 下部バーの合計kcalが見やすい
- 「名前」をコピーできる
- 「kcal」をコピーできる
- 「まとめ」をコピーできる
- MealTracker のプラスアルファに、店名とkcalだけで入力できる

## 実食テスト

- スシローで1回使う
- くら寿司で1回使う
- 店内の通信状況で問題なく表示される
- 店舗を変えたとき、前の店のカートと混ざらない

## 判定

- 問題なければ Vol.0.3 完了
- 使いにくい箇所があれば、Vol.0.3.1 として小修正
